# Browser Extension

This is a browser extension.

## Files

*   `manifest.json`: The extension manifest.
*   `popup.html`, `popup.js`, `popup.css`: The popup UI.
*   `content.js`, `content.css`: The content script.
*   `background.js`: The background script.
*   `detached_player.html`, `detached_player.js`: A detached player UI.
*   `icons/`: Extension icons.
